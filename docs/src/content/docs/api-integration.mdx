Live Data (Online)
- The app imports `WebPortal` + `LoginError` from the `jsjiit` CDN ESM module:
  - `https://cdn.jsdelivr.net/npm/jsjiit@0.0.25/dist/jsjiit.esm.js`
- Login is attempted with `localStorage.username/password` on boot.

Offline Data
- <GitHubFileLink path = "src/components/scripts/artificialW.js" label = "ArtificialWebPortal" /> reads from the local cache via helpers in <GitHubFileLink path = "src/components/scripts/cache.js" label = "Cache.js"/>
- Only data previously cached will be available.


Fee Serialization
- <GitHubFileLink path = "src/lib/jiitCrypto.js" label = "serialize_payload_()" />  encrypts the payload with AES-CBC per-day key schedule.
